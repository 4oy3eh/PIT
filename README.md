# PIT
Programmatūras izstrādes tehnoloģija
Apmeklejot tēva offisu, autors apdomāja par iespēju atvieglot citiem sava laika organizēšanu. Organizācijas sistēma ir diezgan vienkārša, tā parādīta 1. attēlā, ar vieninieku iezīmē lielu papīra skaitu, uz kuriem tiek rakstīti plāni vai uzdevumi, kā arī datumi un piezīmes tiem. Un nākotnē viss tiek sakārtots pašas veidotā kalendārā, kas apzīmēts ar divnieku.
 
1.att.
Lai atvieglotu visu procesu, varetu izveidot analizatoru, kas sadalīs piezīmes notikumos un automātiski piešķirs sevi kalendāram.
Šāda projekta galvenie uzdevumi būs izveidot saikni starp programmu un kalendāru(Google Calendar API), kā arī izveidot vai atrast algoritmu cilvēka rokraksta analizatoram(Optical Character Recognition).
